🚀 Angelina Telidevara - Portfolio Website 🌐

🎨 Live Preview: https://angelinavincent.github.io/

Welcome to my personal portfolio repository! This project showcases my skills, experience, and projects in software development, data engineering, and web technologies. The website is designed to be responsive, interactive, and visually appealing. 


📌 Project Overview
This portfolio is a representation of my professional journey, including:
- My technical skills (Data Engineering, Software Development, Web Development).
- Projects I've worked on, highlighting my expertise in various technologies.
- Work experience, showcasing my roles and contributions in different organizations.
- Contact Form, allowing visitors to reach out to me directly.


🛠️ Technologies Used
🔹 Frontend
- HTML5, CSS3, JavaScript (for interactivity)
- Bootstrap for a responsive layout
- FontAwesome & IonIcons for modern UI elements

🔹 Backend & Other Technologies
- JavaScript & jQuery for smooth animations
- EmailJS for contact form integration
- Google Fonts for better typography
- SweetAlert2 for pop-up alerts

🔹 Hosting & Deployment
- Deployed on GitHub Pages 


🚀 Features
✅ Fully Responsive - Works seamlessly on desktop, tablets, and mobile devices.  
✅ Smooth Animations - Interactive scrolling and hover effects for better UI experience.  
✅ Custom Contact Form - Users can send messages directly via integrated email service.  
✅ Dynamic Content - Includes animated text transitions using `Typed.js`.  
✅ Modular & Scalable - Clean and structured code, easy for enhancements.


💡 How to Use
1. Clone this repository  
   bash
   git clone https://github.com/AngelinaTelidevara/portfolio.git
2. Open `index.html` in any modern browser  
3. To modify content, edit `index.html`, `style.css`, and `main.js`  
4. Deploy using GitHub Pages, Vercel, or Netlify
